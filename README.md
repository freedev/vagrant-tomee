# vagrant-tomee

This Vagrantfile create an Linux VM with Apache TomEE. 

## Supported Platforms

- Debian, Ubuntu (OpenJDK, Oracle)
- CentOS 6+, Red Hat 6+, Fedora

Used with: Centos (6.2, 6.5) and Ubuntu (14.04)

### Provisioning

The provisioning process is based on Chef. 

- tomee
- openssl
- java

### Quick start

Download Vagrant from: https://downloads.chef.io/chef-dk/
